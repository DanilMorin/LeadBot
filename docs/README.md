# Сценарий получения заявок

1. getLastLeads() — получить последние заявки вообще
2. getLeadsByStatus() — получить заявки с конкретным статусом
3. updateStatus() — обновить статус
4. createLead() — создать заявку
   `lead.service.ts` становится центром работы с заявками.

### Commands
Файлы в commands отвечают за прямые команды:
- /start
- /cancel
- /leads
- /newleads
- /export
  
#### Файлы в handlers отвечают за сценарии и события:

- пользователь пишет текст;
- админ нажимает inline-кнопку;
- бот продолжает диалог.

  
#### Файл lead.service.ts отвечает за работу с заявками:

- создать;
- получить;
